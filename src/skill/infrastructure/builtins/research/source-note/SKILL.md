name: source-note
description: Write concise Deep Research source records
source: builtin

Use this skill when capturing a web, docs, or files source for a Deep Research topic.

Workflow:
1. Choose the right source type: web, docs, or files.
2. Assign the matching ID prefix: `W`, `D`, or `F`.
3. Fill the source fields in the exact schema order.
4. Keep excerpts short and cite only what supports related findings.
5. Note reliability, freshness, access limits, and follow-up checks.

Reference files:
- `references/source-note-schema.md` for source note fields, paths, and ID rules.
