# Source Note

Use this skill to capture a web, docs, or files source for a Deep Research topic.

## Workflow

1. Choose the source type: web, docs, or files.
2. Assign the matching ID prefix: `W`, `D`, or `F`.
3. Fill the source fields in the exact schema order.
4. Keep excerpts short and cite only what supports related findings.
5. Note reliability, freshness, access limits, and follow-up checks.

## Guardrails

- Do not create skills.
- Do not write `.ncoworker/skills/**`.
- Do not fabricate sources, citations, titles, URIs, paths, or retrieval details.
