name: finding-synthesis
description: Turn Deep Research notes into cited findings
source: builtin

Use this skill when converting source notes into durable findings.

Workflow:
1. Start from source notes, not memory alone.
2. Write one testable claim per finding.
3. Set scope before confidence so limits are explicit.
4. Cite source IDs in Evidence and keep caveats in Notes.
5. Leave unresolved or weakly supported points in open questions.

Reference files:
- `references/finding-quality.md` for Claim, Evidence, and Confidence guidance.
